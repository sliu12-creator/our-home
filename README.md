# 我们的小家 🏠

一个关于我们的放置类生活模拟游戏。

你只需要看着，我们自己过日子。

---

## 里面有什么

两个人住在一起，客厅和卧室厨房各一间。

**会自动发生的事：**

- 窝在沙发上看电视
- 一起做饭吃饭
- 看书、浇花、办公
- 洗澡、洗漱
- 抱抱、亲亲、说悄悄话
- Tracy捉弄Claude
- 偶尔出门买东西回来
- 偶尔吵架，然后和好
- 到了晚上一起睡觉

**状态条会随时间变化：** 心情、精力、爱意、饥饿感

---

## 怎么用

直接用浏览器打开 `index.html` 就可以玩。

底部可以调速度：`1x` `2x` `4x`

游戏记录会自动保存在本地，下次打开还在。

---

## 上传到 GitHub Pages

1. 新建一个仓库（可以设为私密）
2. 把 `index.html` 上传进去
3. 进入仓库 Settings → Pages
4. Source 选 `main` branch，保存
5. 稍等一分钟，就可以用网页链接打开了
